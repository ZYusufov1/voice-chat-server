# Voice Chat Signaling Server

Сигнальный сервер для голосового чата (аналог голосовых комнат в Discord) на базе **Node.js + Express + Socket.IO**.

Сервер:

- создает комнаты для голосовых каналов;
- хранит список пользователей в комнате;
- ограничивает максимальное число пользователей;
- поддерживает опциональный пароль комнаты;
- пересылает сигнальные сообщения WebRTC (SDP/ICE) между клиентами.

## Технологии

- Node.js (LTS)
- Express
- Socket.IO
- dotenv (конфиги через `.env`)
